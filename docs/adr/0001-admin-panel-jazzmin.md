# ADR 0001: Кастомная админ-панель на Django Jazzmin

## Статус
Принято

## Контекст
Для управления контентом сайта Dastorkon (блюда, категории, сборки, заказы) необходима административная панель. Стандартная Django admin функциональна, но имеет устаревший интерфейс, не соответствующий бренду проекта.

Требования:
- Интерфейс должен отражать визуальный стиль Dastorkon (бордовые акценты, кремовый фон, золотые детали)
- Поддержка мультиязычных моделей (modeltranslation)
- Удобное управление заказами со статусной индикацией
- Простота настройки без написания кастомных шаблонов

## Рассмотренные альтернативы

1. **Стандартная Django admin** — базовая функциональность есть, но интерфейс некастомизируем без шаблонов
2. **django-grappelli** — более современный, но тяжёлый, сложная настройка цветовой схемы
3. **django-jazzmin** — лёгкий, темизация через JSON-конфиг, совместим с django-modeltranslation, активная поддержка
4. **django-unfold** — современный, но требует больше конфигурации и может конфликтовать с modeltranslation

## Решение
Используем **django-jazzmin** как наиболее подходящее решение:
- Конфигурация темы через `JAZZMIN_SETTINGS` без шаблонов
- Поддержка кастомного логотипа, цветов sidebar/header/акцентов
- Совместимость с `adminsortable2` и `modeltranslation` (уже установлены)
- Встроенные UI-компоненты: CollapsibleSidebar, навигационные иконки, поиск
- Лёгкий вес, минимум зависимостей

## Последствия
- `jazzmin` добавляется первой в `INSTALLED_APPS`
- Настройка темы осуществляется в `settings.py` через словарь `JAZZMIN_SETTINGS`
- Цвета интерфейса привязаны к дизайн-токенам Dastorkon:
  - Sidebar: `#721C1C` (brand-red), активный пункт: `#D4AF37` (brand-gold)
  - Top header: `#4A1010` (brand-redDark)
  - Фон контента: `#F5F2EB` (brand-cream)
  - Текст: `#2B1E17` (brand-dark)
  - Акцентные кнопки: `#D4AF37` с hover `#c9a96e`
- Логотип админки копируется из `frontend/public/logo-2.png`
